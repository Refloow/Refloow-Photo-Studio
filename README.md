# Refloow-Photo-Studio
Powerful local easy to use image editor, quick edits, cutting, filters, few click experience right at your desktop

- No watermarks
- No ads
- Free forever
- No logins
- No data collection
- Fully private
- Instant background removal
- Easy image cropping
- Easy image mirroring
- Edit contrast, saturation, brightness & warmth
- Apply one of 40 available professional filters
- Overlay photos and manage layouts for complex photo edits
- Easilly undo and redo with advanced history feature (ctrl Z & ctrl Y)
